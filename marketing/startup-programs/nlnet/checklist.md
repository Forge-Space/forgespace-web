## Key Framing Notes

1. **Lead with MCP standard** — position mcp-gateway as open infrastructure for an open standard
   (analogous to nginx/Caddy for HTTP)
2. **Emphasize self-hostable** — NLnet values non-vendor-dependent solutions
3. **Show existing traction** — 834 users, 9 repos, 2,994+ tests, production deployment
4. **Budget justification** — each line item maps to a concrete deliverable
5. **Open governance model** — rules as data files, community-contributable
